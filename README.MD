🧠 Job Application Agent (WIP)

A Stateful Multi-Agent System that analyzes a resume against a job description and improves it iteratively.

---

🚀 Current Idea

- Extract key requirements from Job Description
- Compare with resume using embeddings
- Calculate similarity score
- Improve resume using AI agents

---

🤖 Planned Agents

- Analyst → Extracts structured data
- Matcher → Computes similarity + missing skills
- Optimizer → Rewrites resume bullets
- Critic → Evaluates and iterates

---

🛠️ Tech (Planned)

- LangGraph
- FastAPI
- Gemini / LLMs
- Vector Embeddings
